# CareLink

CareLink is a Flask-based web application developed to provide a secure, user-friendly, and scalable platform for managing services and user interactions. The application implements authentication, database connectivity, and environment-based configuration while following best practices in backend web development.

## Key Features
- Secure user authentication and authorization
- Database integration for efficient data management
- Environment variable configuration using `.env`
- Organized and scalable project structure
- Easy deployment and maintenance

## Technology Stack
- Python
- Flask
- MySQL
- HTML, CSS, JavaScript
- JWT Authentication
- Python Dotenv
